Kit Desenvolvimento Internet of Things

Kit 4G v.2

O que é?

O equipamento

Baseado na plataforma de prototipagem eletrônica livre Arduíno, foi desenvolvido um shield com sensores de luminosidade, contato, temperatura e luz.

A linguagem

Uma biblioteca em JavaScript (node.js) foi criada para simplificar o acesso aos sensores pelos desenvolvedores e enviar os dados diretamente a uma plataforma baseada na nuvem.

A nuvem

Os dados são enviados a uma plataforma de alta performance escalável, que foi desenvolvida para ser flexível e modular para facilitar a construção de aplicações M2M baseadas em dados em tempo real.

A conectividade

Utilizando a mais avançada tecnologia 4G, o kit de desenvolvimento envia dados em tempo real para uma nuvem permitindo ao usuário acessá-los de qualquer lugar.
